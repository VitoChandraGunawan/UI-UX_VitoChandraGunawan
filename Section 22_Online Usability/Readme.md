# 22 Online Usability Testing

## Kelebihan dari online Usability Testing :
1. Dapat menemukan user dengan mudah
2. Lebih cepat dan lebih murah
3. Jangkauan user yang sangat luas
4. Efisien

## Persiapan Usability Testing :
1. Lakukan persiapan
2. Usability Testing
3. Mengelola insight Usability Testing
4. Apasih tools yang biasa digunakan untuk online Usability Testing :
5. Figma dan maze

## Tahap persiapan :
1. Membuat clickable prototype di figma
2. Setup Block Skenario di Maze
3. Jalankan dan share link online usability kepada user

## Apasih jenis-jenis Block Skenario?
1. Mission
2. Yes/no
3. Opinion Scale
4. Multiple Choice
5. Open Question
6. Context Screen
7. Tree Test
8. Card Sort
9. 5-Second Test
10. Legal Screen

## Tips membuat mission Usability Testing :
- Mulai dari tujuan yang simpel
- berikan 1 tugas dalam sekali waktu
- Tugas yang diberikan pastikan realistis dan menyerupai kehidupan sebenarnya
- Apasih Discovery Usability Testing? 👨‍💻
- Discovery Usability Testing adalah sebuah metode untuk menggali insight dari user terhadap 1 tampilan sebuah website
- dengan memberikan konteks yang tepat, rule of 5 second impression, berikan pertanyaan.

## Tips melakukan Discovery Usability testing :
1. Buat misi sesimpel mungkin
2. Ikuti user flow yang sudah dirancang
3. Pastikan prototype anda berjalan dengan lancar
4. Hindari bahasa teknis
5. Tentukan parameter
